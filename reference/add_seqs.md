# Add seqs

Add seqs

## Usage

``` r
add_seqs(x, seqs, ...)
```

## Arguments

- x:

  a gggenomes or gggenomes_layout object

- seqs:

  the sequences to add

- ...:

  pass through to
  [`as_seqs()`](https://thackl.github.io/gggenomes/reference/as_seqs.md)

## Value

a gggenomes or gggenomes_layout object with added seqs
