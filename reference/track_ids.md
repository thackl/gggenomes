# Named vector of track ids and types

Named vector of track ids and types

## Usage

``` r
track_ids(x, track_type, ...)
```

## Arguments

- x:

  A gggenomes or gggenomes_layout object

- track_type:

  restrict to any combination of "seqs", "feats" and "links".

- ...:

  unused

## Value

a named vector of track ids and types
