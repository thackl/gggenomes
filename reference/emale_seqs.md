# Sequence index of 6 EMALE genomes (endogenous virophages)

A data set containing the sequence information on 6 endogenous
virophages found in the genomes of the marine protist *Cafeteria
burkhardae*.

## Usage

``` r
emale_seqs
```

## Format

A data frame with 6 rows and 4 columns

- file_id:

  name of the file the data was read from

- seq_id:

  sequence identifier

- seq_desc:

  sequence description

- length:

  length of the sequence
